# Definition of Done

A PRD is done only when all checks below pass:

- required headings exist and are in the expected order
- `requirements.yml` exists and passes structure validation
- `prd.md` does not duplicate FR/AC entries
- no unresolved `TODO`, `TBD`, or `FIXME` markers remain
- assumptions and open questions are explicit
- verification expectations are explicit
- scope boundaries are explicit
